# Module 03 — Acceptable Use Policy

**Training Program:** AKG IT Security Awareness (IT-SEC-AWARE-01)  
**Estimated Read Time:** ~5 minutes

---

## Overview

The AKG Acceptable Use Policy (AUP) defines the boundaries for using company-owned technology, networks, and data. Violations can result in disciplinary action up to and including termination, and may carry legal liability.

---

## Permitted Use

| Activity | Status |
|----------|--------|
| Business use of corporate devices | ✅ Required |
| Limited personal browsing during breaks | ✅ Tolerated |
| Installing software from the Software Center | ✅ Permitted |
| Using VPN for all remote work | ✅ Required |
| Using corporate email for business comms | ✅ Required |

---

## Prohibited Activities

The following are expressly prohibited and subject to disciplinary action:

- Installing unlicensed software, pirated content, or personal applications (gaming, P2P clients)
- Accessing, storing, or transmitting pornographic or offensive content
- Using company resources for personal business or side projects
- Circumventing security controls — disabling antivirus, using unauthorized VPNs, using Tor
- Connecting unauthorized devices (personal USB drives, mobile hotspots) to the corporate network
- Storing AKG data on personal devices, personal Google Drive, or unapproved cloud storage
- Using personal email accounts to send or receive work-related documents

> **Zero Tolerance:** Disabling or attempting to bypass security controls (EDR, DLP, firewall) is a zero-tolerance violation — even when the intent was to improve productivity.

---

## Data Classification

All AKG data must be handled according to its classification level. When in doubt, treat any document as **Internal** until confirmed otherwise.

| Level | Description | Handling |
|-------|-------------|----------|
| **Public** | Approved for external distribution | No restrictions |
| **Internal** | AKG employees only | Do not share externally without authorization |
| **Confidential** | Need-to-know — contracts, financials, HR | Explicit authorization required to share |
| **Restricted** | PII, customer data, security configurations | Requires written approval from manager + IT |

---

## Remote Work

When working outside AKG premises:

- **VPN is mandatory** before accessing any AKG system or resource
- Do not connect to public Wi-Fi without VPN active
- Do not work on confidential or restricted data in public spaces without a privacy screen
- Never leave a laptop unattended in a vehicle or public area

---

## Monitoring Notice

By using AKG systems, you acknowledge that all activity on corporate devices and networks may be monitored, logged, and audited for security and compliance purposes. This includes email, web browsing, file transfers, and application usage.
